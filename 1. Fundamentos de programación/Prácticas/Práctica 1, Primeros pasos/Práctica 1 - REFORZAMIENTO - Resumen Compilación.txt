¿Qué es un compilador?
Is a program that translates a programming language's source code into machine code, bytecode or another programming language.

¿Qué es una máquina virtual?
Is a compute resource that uses software instead of a physical computer to run programs and deploy apps.

¿Qué es una variable de entorno?
Is like independent variables in a mathematical equation, these take on values that change the results of the program.

¿Qué es un depurador?
Is a program that allows detecting and diagnosing bugs for codes or computer programs.
